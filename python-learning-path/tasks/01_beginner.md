# Задания Level 1

1. Добавь в `python_learning.cli` команду `count`, которая выводит число задач.
2. Расширь `Task.priority`: запрети значения вне диапазона `1..5`.
3. Добавь поле `tags: list[str]` в модель `Task`.
4. Обработай ошибку, если пользователь вызывает `done` без ID.
5. Напиши 3 мини-скрипта:
   - подсчет частоты слов в строке;
   - фильтрация четных чисел;
   - безопасное деление (`None` при делении на 0).
