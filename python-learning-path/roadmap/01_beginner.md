# Level 1: Beginner (0 -> уверенный базовый)

## Цель
Понять синтаксис Python и уметь писать простые, читаемые скрипты.

## Темы

- типы данных, условия, циклы
- функции и аргументы
- базовые коллекции (`list`, `dict`, `set`, `tuple`)
- обработка ошибок (`try/except`)
- простая CLI-логика

## Практика в проекте

- разбор `src/python_learning/models.py`
- разбор `src/python_learning/cli.py`
- выполнение заданий из `tasks/01_beginner.md`

## Критерий готовности

- пишешь функции без подсказок
- понимаешь разницу изменяемых/неизменяемых структур
- можешь добавить новую команду в CLI
