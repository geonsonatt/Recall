# Level 3: Middle (рабочий -> системный)

## Цель
Писать поддерживаемый код с типизацией, проверками качества и аналитикой.

## Темы

- typing и статическая проверка (`mypy`)
- quality gates (`ruff`, `pytest`)
- вычисление метрик и агрегаций
- проектирование API модуля
- разделение бизнес-логики и инфраструктуры

## Практика в проекте

- изучи `src/python_learning/analytics.py`
- улучши типизацию в `src/python_learning/service.py`
- выполни `tasks/03_middle.md`
- добавь свои тесты в `tests/test_analytics.py`

## Критерий готовности

- понимаешь и исправляешь mypy/ruff замечания
- можешь аргументировать архитектурное решение
- пишешь тесты на граничные условия
