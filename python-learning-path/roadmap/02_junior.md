# Level 2: Junior (база -> рабочий developer)

## Цель
Собирать небольшие приложения из нескольких модулей и покрывать их тестами.

## Темы

- `dataclass` и модель предметной области
- работа с файлами и `pathlib`
- модульная архитектура
- unit-тесты (`pytest`)
- рефакторинг без изменения поведения

## Практика в проекте

- изучи `src/python_learning/storage.py`
- изучи `src/python_learning/service.py`
- прогоняй `tests/test_storage.py` и `tests/test_service.py`
- выполни `tasks/02_junior.md`

## Критерий готовности

- проект разделен на слои (модель/хранение/сервис)
- умеешь писать тесты на happy-path и edge cases
- можешь поменять формат хранения без поломки сервисного слоя
